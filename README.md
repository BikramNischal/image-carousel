# OR

## Run locally


1. clone the repo by

```
git clone https://github.com/BikramNischal/image-carousel.git
```

2. change to project directory
```
cd image-carousel
```

3. install dependencies
```
npm install 
```

4. build the project
```
npm run build
```

5. run the project
```
npm run dev
```
